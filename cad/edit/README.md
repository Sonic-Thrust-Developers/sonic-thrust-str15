# 📐 STR15 Editable CAD Models (`/edit`)

This folder contains the master, editable source CAD files for the STR15 70mm EDF rotor and housing.

## 📂 File Inventory

*   `str15-rotor-master.dwg` - Master 3D AutoCAD drawing for the high-performance rotor.
*   `str15-housing-master.dwg` - Master 3D AutoCAD drawing for the outer housing.
*   *(Optional)* `.step` / `.iges` - Neutral CAD formats for cross-compatibility with Fusion 360, SolidWorks, or Inventor.

## 🛠️ Contribution Guidelines

If you are modifying these designs:
1. **Maintain Tolerances:** Do not alter the core motor shaft sleeve diameter or outer blade tip clearance without testing dynamic balance.
2. **Keep History:** Ensure your AutoCAD layers are clean and labeled before submitting a Pull Request.
3. **Drafting Standards:** Work in metric (millimeters) as the standard unit of measurement for this project.
