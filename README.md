# sonic-thrust-str15
Open-source high-performance 70mm EDF rotor and housing replacement engineering.
## 🎨 Logo & Branding Contributions
Want to improve the Sonic Thrust visual identity? We welcome community logo and decal designs! 
* To submit a new logo design or variation, please upload it to the `assets/` folder via a **Pull Request**.
* All submissions will be reviewed by the core team. 
* Approved designs may be adopted as the official project logo or featured as alternative community decal options in the repository!
